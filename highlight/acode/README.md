# PyScript - Syntax Highlighting - Acode
_Plugin for PyScript 1.13.0+_

This is a syntax highlighting for PyScript programming language! See more about PyScript
[here](https://github.com/azzammuhyala/pyscript). For plugin source code, you can see
[here](https://github.com/azzammuhyala/pyscript/tree/main/highlight/acode).

> NOTE: This plugin has been rejected by Acode because it uses the Ace Editor engine, which they abandoned and switched
to CodeMirror. I've tried to adapt it to CodeMirror compatibility, but unfortunately, i'm running into a lot of
debugging issues. I've decided to discontinue this project (_i think is not_). You can download the last final zip file
from this [GitHub](https://github.com/azzammuhyala/pyscript/releases/download/v1.13.3/plugin.zip).
