# Change Log

## [1.3.2] - 08/04/2026

### Added
- Operator `<>` (added in PyScript 1.12.10).
- Builtins `infj`, `nanj`, and `frozendict` (`frozendict` is only for Python 3.15+ versions).
- File icon for `*.pys` and `*.pyscript_history`.

### Fixed
- Fixed some bugs.
- New PyScript logo.

### Removed
- Builtins `infinity`, `Infinity`, `nonanumber`, `NaN`, `NotANumber`, and `ellipsis` (removed in PyScript 1.12.10).
- Keywords `def` and `define` (removed in PyScript 1.12.10).
