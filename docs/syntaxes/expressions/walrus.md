# Walrus
Walrus adalah operasi penugasan dimana ia menetapkan nilai variable sebagai bagian dari ekspresi. Walrus masuk kedalam assignment yang berati dapat menetapkan nilai berupa identifier, attribut, ataupun subscript. Walrus tidak dapat meunpack nilai variable sekaligus layaknya menetapkan nilai variable biasa.

Berikut ini adalah salah satu penggunaan walrus dalam kode:
<pre><span style="color:#C586C0">if</span><span style="color:#D4D4D4"> </span><span style="color:#FFD705">(</span><span style="color:#D45DBA">(</span><span style="color:#8CDCFE">x</span><span style="color:#D4D4D4"> := </span><span style="color:#CE9178">&#x27;go vegan&#x27;</span><span style="color:#D45DBA">)</span><span style="color:#D4D4D4">.</span><span style="color:#DCDCAA">strip</span><span style="color:#D45DBA">()</span><span style="color:#D4D4D4">.</span><span style="color:#DCDCAA">lower</span><span style="color:#D45DBA">()</span><span style="color:#D4D4D4"> == </span><span style="color:#CE9178">&#x27;go vegan&#x27;</span><span style="color:#FFD705">)</span><span style="color:#D4D4D4"> </span><span style="color:#FFD705">{</span><br><span style="color:#D4D4D4">    </span><span style="color:#DCDCAA">print</span><span style="color:#D45DBA">(</span><span style="color:#CE9178">&quot;sybau&quot;</span><span style="color:#D45DBA">)</span><br><span style="color:#FFD705">}</span><span style="color:#D4D4D4"> </span><span style="color:#C586C0">else</span><span style="color:#D4D4D4"> </span><span style="color:#FFD705">{</span><br><span style="color:#D4D4D4">    </span><span style="color:#DCDCAA">print</span><span style="color:#D45DBA">(</span><span style="color:#CE9178">&quot;don&#x27;t sybau&quot;</span><span style="color:#D45DBA">)</span><br><span style="color:#FFD705">}</span></pre>
