# Change Log

## [1.11.1] - 05/02/2026

- Loaded modules will be removed in `modules` / `sys.modules` when an exception occurs during import.
- PyScript Editor code and also new arguments (`-e`, `--editor`) for it.

- **Fixed some bugs**.
